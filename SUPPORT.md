# Support

If you want to use or extend Finance Journal, start here:

1. `README.md`
2. `finance-journal-orchestrator/references/command-cheatsheet.md`
3. `finance-journal-orchestrator/references/openclaw-skill-functional-spec.md`
4. `GIT_SYNC_WORKFLOW.md`

If you believe you found a defect or want to suggest a feature:

- use the bug report template for defects
- use the feature request template for product or workflow ideas

If the issue is security-related, do not open a public issue. Follow `SECURITY.md` instead.

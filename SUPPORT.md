# Support

如果你想使用或扩展这个项目，建议按下面顺序获取帮助：

1. 先看 `README.md`
2. 再看 `finance-journal-orchestrator/references/command-cheatsheet.md`
3. 行为/会话问题查看 `finance-journal-orchestrator/references/openclaw-skill-functional-spec.md`
4. Git 与账本同步问题查看 `GIT_SYNC_WORKFLOW.md`

如果确认是缺陷或功能建议：

- 缺陷：请使用 GitHub bug issue 模板
- 新功能：请使用 GitHub feature request 模板

如果是安全问题，请不要公开提 issue，改走 `SECURITY.md`。
